Quantum-Hybrid Graph Architectures for High-Luminosity LHC Tracking
Solving the Combinatorial Explosion in Particle Reconstruction
📄 Paper (PDF) 💻 Code (GitHub)

Problem Statement

Context: The High-Luminosity Large Hadron Collider (HL-LHC) will increase collision rates to 40 MHz.
The Challenge: Each event produces ~100,000 discrete 3D sensor hits (x, y, z) from ~10,000 particles. Reconnecting these dots into continuous tracks creates a combinatorial explosion (O(N!) complexity).
The Constraint: Current algorithms achieve high accuracy but are too slow (minutes/event) for real-time hardware triggers (microseconds required).

Methodology & Evolution

Phase 1: Classical Baseline (DBSCAN)
Approach: Used standard density-based clustering (DBSCAN) after conformal mapping.
Result: Failed. Efficiency ~7.75%.
Insight: Heuristic clustering cannot handle crossing tracks in dense environments.

Phase 2: State-of-the-Art Reproduction (Iterative Seeding)
Approach: Implemented the "Top-Quarks" (Kaggle Winner) logic using KD-Tree spatial indexing and iterative 3-point Helix Fitting.
Result: Success. Achieved 98.95% ± 0.12% Efficiency across 20 benchmark events.
The "Bottleneck": Average processing time was 270 seconds per event (serial execution). This proves classical methods are accurate but computationally impossible for HL-LHC triggers.

Phase 3: The Novel Solution (Hybrid Quantum-GNN)
Concept: Replaced serial fitting with a parallelizable "Filter & Solve" pipeline.
Step A (Filter): A Graph Neural Network (MLP/GNN) classifies valid connections, achieving 0.9980 ROC-AUC locally but leaving global conflicts (bifurcations).
Step B (Solver): Formulated the conflict resolution as a Quadratic Unconstrained Binary Optimization (QUBO) problem, enforcing "Gauss's Law" (flow conservation) constraints.
Result: Simulated Quantum Annealing achieved 99.1% Purity with a total processing time of ~25 seconds (10x speedup in Python, theoretically O(1) depth on Quantum Hardware).

Conclusion

Classical geometric algorithms have hit a speed ceiling.
Our Hybrid architecture demonstrates that we can trade a negligible amount of accuracy (<1%) for a massive gain in computational speed and parallelizability.
This establishes a viable path for using Quantum-AI hybrids in the future HL-LHC Level-1 Trigger.
